# unix_profile
My ZSH & VIM profiles. 

### Based on 
- [oh-my-zsh](https://github.com/robbyrussell/oh-my-zsh)
- [zsh-autosuggestions](https://github.com/zsh-users/zsh-autosuggestions.git)

### Setup and Install
```bash
git clone https://github.com/jodydadescott/jodys_zsh.git ~/.zshprofile && cd ~/.zshprofile
sh setup
```
